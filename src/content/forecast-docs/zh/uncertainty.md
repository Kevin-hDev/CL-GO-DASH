# 不确定性

预测从来都不是绝对真理。因此 Forecast 会显示一个中心值和一个不确定性区间，以表明轨迹周围的风险。

## 中心值

中心值是模型的主要估计。

示例：

```text
2026-06-01 -> 预测 142 单订单
```

它代表根据所用数据得出的最可能的情景。

## 不确定性区间

区间表示围绕中心值的可能范围。

示例：

```text
预测值：142
下限：128
上限：157
```

简单解读：模型估计为 142，但认为 128 到 157 之间的值仍然合理。

## 分位数

模型可以返回分位数。

| 字段 | 含义 |
| --- | --- |
| q10 | 可能的低值 |
| q50 | 中心值或中位数 |
| q90 | 可能的高值 |

q10 与 q90 之间的差距越大，模型越不确定。

## 为什么不确定性会增加

当出现以下情况时，不确定性可能会增加：

- 历史数据较短；
- 目标波动剧烈；
- 预测期较长；
- 缺少上下文变量；
- 数据中出现近期突变；
- 可能存在多种未来情景。

## 正确用法

中心值用于判断趋势。

不确定性区间用于判断风险。

严肃的决策必须同时看两者。
